//
// "train_test_split.py" の使い方
//

本Pythonプログラムは、画像とYOLOのアノテーションデータ(*.txtファイル)を、学習用と検証用にランダムに
別けてくれる。


■使い方
　1．以下のように本プログラム、画像、アノテーションデータを配置する。
　
　　.\train_test_split.py			# 本プログラム
　　.\images						# 画像ファイルを格納したフォルダ
　　.\bbox_txt						# アノテーションデータ(*.txt)を格納したフォルダ

　2．プロンプトにて以下のように入力して実行。
　
　　python train_test_split.py


■備考
　・学習用と検証用の比率は、プログラムのコードを変更することによって可能。
